Html References

Basic Tags
Tag Name Description
<!DOCTYPE html> !DOCTYPE Defines the document type and it's a self closing tag.
<html></html> Html The html tag represents the root of an HTML document.
<head></head> Head Contains metadata/information for the document.
<title></title> Title Defines a title for the document.
<body></body> Body Defines the document's body.
<h1></h1> to <h6></h6> Headings Defines HTML headingsand it ranges from h1 to h6.
<p></p> Paragraph Defines a paragraph.
<br> Break Used to inserts a single line break and its self closing.
<hr> Horizontal It's often used to displayed a horizontal line that is used to separate content in an HTML page.
<col> Col Specifies column properties for each column within a <col> element.
<!-- --> Comments Defines a comment.
Table Tags
Tag Name Description
<table></table> Table The wrapper element for all HTML tables.
<thead></thead> Table Head The set of rows defining the column headers in a table.
<th></th> Table header Th stands for table header.
<tbody></tbody> Table Body The set of rows containing actual table data.
<tr></tr> Table Row The table row container.
<td></td> Table Data The table row container.
<tfoot></tfoot> Table Footer The set of rows defining the footer in a table.
<caption></caption> Caption The Caption defines a table caption.
<colgroup></colgroup> Colgroup Specifies a group of one or more columns in a table for formatting.
<col> Col Specifies column properties for each column within a <colgroup>, and it's self closing.
Formatting Tags
Tag Name Description
<abbr></abbr> Abbreviation Defines an abbreviation or an acronym
<address></address> Address Defines contact information for the author/owner of a document/article
<b></b> Bold Defines bold text
<bdi><bdi> Bi-Directional Isolation Isolates a part of text that might be formatted in a different direction from other text outside it
<bdo><bdo> Bi-Directional Override Overrides the current text direction
<blockquote></blockquote> Blockquote Defines a section that is quoted from another source
<cite></cite> Cite Defines the title of a work
<cold></cold> Code Defines a piece of computer code
